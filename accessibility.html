Accessibility Project

Activities

From research to refined user journeys, the project focused on uncovering and shaping new opportunities for a better user experience.

Date

May 2025

Brief & process

Accessibility Project Buddhify website

The accessibility phase was divided into two parts to allow for a deeper analysis.

  1. Create user personas with disabilities to ensure inclusivity
  2. Develop use cases to explore potential scenarios
  3. Map the current site’s user journey to identify new opportunities
  4. Rationalize the opportunities that emerged from the analysis
  5. Redesign the user journeys based on the improved structure

Accessibility Project Part 1

User Persona and User Journey

Analysis of five pages of the Buddhify website (Home, The App, Become a Member, Blog, and Brand Resources).

The evaluation was carried out using the four WCAG principles; Perceivable, Operable, Understandable, and Robust, as a reference framework.

This analysis highlighted issues and potential solutions that will be addressed in the next phase.

Accessibility Project Part 1

Pain points and opportunities

From the study of the user journeys of three personas, each with a different disability, I was able to identify the pain points and opportunities on the current Buddhify website.

For example: complex and confusing navigation, lack of personalization to streamline the experience for users experiencing stress, and difficulties in color perception for users with color blindness..

Pain Points

The website analysis revealed the pain points that need improvement.

Opportunities identified

The opportunities identified allowed for the improvement of the website’s accessibility.

Accessibility Project Part 2

Accessibility Project Part 2

Key issues identified and potential solutions

From the analysis of five pages of the Buddhify website, I identified several general issues: insufficient visual contrast, unclear visual hierarchy, and inconsistencies in links and calls-to-action, all of which hinder usability and navigation.

From these main issues, potential solutions have emerged, which I will analyze and modify later.
